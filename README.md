# World of Warcraft Item ID/Name Tables

## What are these tables?

This repository contains Lua tables that pair World of Warcraft item IDs and item names.

## Locales

- deDE
- enUS, enGB
- esES
- esMX
- frFR
- itIT
- koKR
- ptBR, ptPT
- ruRU
- zhCN
- zhTW
